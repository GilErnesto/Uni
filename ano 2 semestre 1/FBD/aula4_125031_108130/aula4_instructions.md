## Regras para submissão do guião
Guide submission rules

### Perguntas
Questions

Responda a cada uma das questões dentro da zona de *bloco de codigo* imediatamente abaixo da respectiva pergunta. Os *blocos de codigo* são delimitados por três "`" em linhas isoladas. 
Continua a ser encorajado a escrever comentários nas suas respostas. Os comentários devem ser incluidos no *bloco de codigo* da respectiva pergunta. Garanta que os comentários estão devidamente 'escaped'.

Em algumas perguntas é necessário submeter uma imagem.
Neste caso, o template terá um link para a imagem: ex. `![ex_2_2!](ex_2_2.png "AnImage")`. Deve garantir que tem o ficheiro `ex_2_2.png` na mesma pasta do template. Além do formato `.png` pode também submeter uma imagem nos seguintes formatos: `.png`, `.jpg`, `.bmp`, `.svg`, e `pdf`. Neste caso está autorizado a mudar o link no template para a extensão correcta. Por exemplo `![ex_2_2!](ex_2_2.svg "AnImage")`. **Atenção! Apenas pode mudar a extensão do ficheiro. Nunca o nome, nem outras partes do link**.

Em outras perguntas terá de entregar um ficheiro SQL.
Nestes casos, existirá no template um link com este aspecto: `[Some link Text](ex_4_1_3.sql "SQLFileQuestion")`. Nestas perguntas basta garantir que existe um ficheiro SQL com o mesmo nome do link na mesma pasta do template. Neste exemplo, o ficheiro necessario seria `ex_4_1_3.sql`. **Atenção! Não mudar nada nestes links.**

**Não altere nenhum elemento do template** a não ser dentro dos respectivos *blocos de codigo* e a extensão das imagens se necessário. Caso contrário, a sua submissão será invalidada.

---

Answer each of the questions within the *code block* zone immediately below the respective question. *code blocks* are delimited by three "`" on isolated lines.
You are still encouraged to leave comments on your responses. Comments must be included in the *code block* of the respective question. Ensure comments are properly 'escaped'.

In some questions it is necessary to submit an image.
In this case, the template will include a link to the image: ex. `![ex_2_2!](ex_2_2.png "AnImage")`. You must ensure that you have the `ex_2_2.png` file in the same folder as the template. In addition to the `.png` format you can also submit an image in the following formats: `.png`, `.jpg`, `.bmp`, `.svg`, and `pdf`. In this case you are allowed to change the link in the template to the correct extension. For example `![ex_2_2!](ex_2_2.svg "AnImage")`. **Attention! You can only change the file extension. Never the name, nor other parts of the link**.

In other questions you will have to submit an SQL file.
In these cases, there will be a link in the template that looks like this: `[Some link Text](ex_4_1_3.sql "SQLFileQuestion")`. In these questions, just ensure that there is an SQL file with the same name as the link in the same folder as the template. In this example, the required file would be `ex_4_1_3.sql`. **Attention! Do not change anything in these links.**

**Do not change any elements of the template** except within the respective *code blocks* and the extension of the images if necessary. Otherwise, your submission will be invalidated.
